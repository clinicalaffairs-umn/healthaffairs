# THEME-INSTALL.md

## LAST REVIEWED
Last tested and working on Node version 14.17.0 and NPM version 6.14.13 on 2021-6-11, by Adam. You can use NVM to switch to these versions if yours are different and not working.

```
nvm install 14
nvm use 14
```

## ABOUT OACA_SUBTHEME THEME

For Drupal 9, oaca_subtheme uses the Folwell UMN theme starting place, and is meant to expand on that to theme OACA sites

## GETTING STARTED

Working with oaca_subtheme requires a few things: 

To get started:
(a) get a local site instance spun up following the instructions in the site/default repo.
(b) Make sure you have node and npm installed globally and that you are using the correct versions of Node and NPM outlined by the note in the last reviewed section of this doc. 
(d) cd into the folder for oaca_subtheme
(e) run: 
```
npm install
```

Working: 
(a) cd into the oaca_subtheme folder
(b) run: 
```
npm start
```
